# 🚗 EasyCarS - Car Dealership Website

## 📌 Project Overview

**EasyCarS** is a modern, responsive car dealership website designed to showcase premium vehicles, provide inventory browsing, and deliver a smooth user experience with animations and interactive UI elements.

This project is built using **HTML, CSS, and JavaScript** and focuses on frontend design and user interaction.

---

## ✨ Features

* 🚘 **Hero Carousel Slider**

  * Dynamic car showcase with smooth animations
  * Auto-slide and manual navigation (Next/Prev buttons)

* 📦 **Inventory Section**

  * Displays available cars with details like:

    * Price
    * Fuel type
    * Transmission
    * Mileage

* 🎯 **Interactive UI Animations**

  * Scroll-based animations using Intersection Observer
  * Smooth transitions for cards and sections

* 🛒 **Buy & Sell Section**

  * Dedicated sections for buying and selling cars

* 🏷️ **Brand Showcase**

  * Sliding brand logos animation

* 📰 **Blog Section**

  * Latest automotive blog previews

* 📍 **Map Section**

  * Location display (static image)

* 📱 **Responsive Design**

  * Works across desktop and mobile devices

---

## 🛠️ Technologies Used

## ⚙️ How It Works

### 🔹 Carousel Functionality

* Implemented using JavaScript DOM manipulation
* Automatically cycles through slides every few seconds
* Manual navigation supported

### 🔹 Scroll Animations

* Uses **Intersection Observer API**
* Cards and sections appear with animation when visible

### 🔹 Responsive Layout

* Flexbox and Grid used for layout design
* Media queries for mobile responsiveness
